Sey Good Morning - Breakfast Delivery Website

A modern, responsive website for Sey Good Morning - a fresh breakfast delivery service in Accra, Ghana.

## About This Project

This project was initially founded using a template from [Lovable.dev](https://lovable.dev), and has been extensively developed and customized to create a fully functional breakfast delivery website with shopping cart functionality, order placement system, and responsive design.

 Features

- **Shopping Cart System** - Add items to cart with quantity management
- **Fully Responsive** - Optimized for mobile, tablet, and desktop
- **Modern UI** - Built with shadcn/ui and Tailwind CSS
- **Smooth Navigation** - Client-side routing with scroll restoration
- **Menu Pages** - Interactive menu with pagination
- **Contact Form** - Get in touch with the business
- **Special Offers** - Display promotional codes and discounts
- **Fast Performance** - Built with Vite for optimal loading speeds

Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **React Router** - Client-side routing
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI component library
- **TanStack Query** - Data fetching
- **Sonner** - Toast notifications
